# AirdropX MATLAB All Package Notes

This archive contains the MATLAB/Simulink-related parts of AirdropX.

## Included Runtime Files

- `setup_airdropx_simulink.m`: project-root MATLAB setup shortcut.
- `matlab/untitled1.slx`: Simulink model.
- `matlab/*.m`: setup, simulation parameters, controller blocks, CARP/CEP logic, metrics, plotting, report, video, release check, and CSV export scripts.
- `matlab/vr/`: MATLAB VR helper files, if present.
- `matlab/sfunc_jsbsim/`: JSBSim S-Function source, build script, README, and bundled Windows x64 MEX file.
- `aircraft/`: JSBSim aircraft and initial-condition files used by the model.
- `engine/`: JSBSim engine/thruster files required by the bundled MQ-9 aircraft model.
- `third_party/JSBSim/`: JSBSim source tree.
- `third_party/jsbsim-win64/`: Windows x64 JSBSim headers and libraries used to rebuild the MEX file.
- `third_party/build_jsbsim_win64.ps1`: helper script for rebuilding the Windows JSBSim package.
- MATLAB-related documentation and acceptance/report files.

## Excluded Generated Files

The archive intentionally excludes generated/cache/output files:

- `slprj/`
- `matlab/slprj/`
- `matlab/results/`
- `matlab/outputs/`
- `matlab_tmp/`
- `matlab_clean_start/`
- `*.slxc`
- `*.slx.autosave`
- Python cache and virtual environments

These files are regenerated by MATLAB/Simulink when needed.

## Quick Test After Extracting

Open MATLAB, then run:

```matlab
cd('path/to/AirdropX_matlab_all/AirdropX')
addpath('matlab')
airdropx_release_check
```

To run a smoke simulation and export CSV:

```matlab
r = airdropx_run_and_export("RunName", "download_test");
```

Output CSV files will be written under:

```text
matlab/results/download_test/
```

## MATLAB Runtime Requirements

- MATLAB with Simulink.
- Windows 10/11 x64 is the ready-to-run target for the bundled `mexw64`.
- The bundled S-Function is `matlab/sfunc_jsbsim/sfun_airdropx_jsbsim.mexw64`.
- Other operating systems or incompatible MATLAB/compiler combinations require rebuilding the S-Function.

## Rebuild Requirements

For Windows x64:

- MATLAB with a supported C++ MEX compiler configured by `mex -setup C++`.
- A C++17-capable compiler supported by your MATLAB release, such as Microsoft Visual Studio Build Tools.
- Bundled JSBSim headers and libraries under `third_party/jsbsim-win64/`.

Rebuild from MATLAB:

```matlab
cd('path/to/AirdropX_matlab_all/AirdropX')
addpath('matlab')
addpath('matlab/sfunc_jsbsim')
clear mex
build_sfun_airdropx_jsbsim
```

If the bundled JSBSim library is incompatible, rebuild JSBSim first:

```powershell
cd path\to\AirdropX_matlab_all\AirdropX\third_party
powershell -ExecutionPolicy Bypass -File .\build_jsbsim_win64.ps1
```

Then pass the rebuilt install root to MATLAB:

```matlab
build_sfun_airdropx_jsbsim("C:\path\to\jsbsim-install")
```
