## 4 TC-B1：坐标系契约

### 4.1 坐标系定义

本项目采用局部 NED（北-东-下）坐标系：

- **N 轴（北向）**：飞机飞行方向（heading=0°），单位 m
- **E 轴（东向）**：垂直于飞行方向，单位 m
- **原点**：飞机仿真起始位置

### 4.2 禁止行为（coordinate_contract.md）

| 禁止行为 | 说明 |
|----------|------|
| 后验改靶 | 禁止把 Target_N 改成 Impact_N |
| 释放点不补偿 | 禁止 Release_N = Target_N（不做弹道补偿） |
| CEP 口径混淆 | 禁止用 CEP_around_mean 替代 CEP_to_target |

### 4.3 impact_metrics.py

统一 CEP 计算接口，强制以固定目标点为圆心：

```python
# 正确：以固定目标点为圆心
CEP50_to_target = percentile(dist_to_target, 50)

# 禁止：以落点均值为圆心
CEP50_around_mean = percentile(dist_to_mean, 50)  # 不得用于验收
```

### 5.3 k_drag 标定

**注意**：正值表示阻力使货物减速（弹道距离 < 理论值），与物理直觉一致。

## 6 TC-B3：CARP 释放点求解器

### 6.1 求解原理

给定固定目标点 Target_N，反算飞机应在何处释放货物：

$$Release_N = Target_N - D_{ballistic} - D_{wind} - D_{delay}$$

其中：

$$D_{ballistic} = v_N \cdot t_{fall} - k_{drag} \cdot v_N^2 \cdot t_{fall}$$

$$t_{fall} = \sqrt{\frac{2H}{g}}$$

### 6.2 单元测试结果

|----------|-----------|-------------|--------|------|
| 100m | -50.7649m | 150.7649m | -150.7649m | ✅ PASS |
| 150m | -0.7649m | 150.7649m | -150.7649m | ✅ PASS |

## 8 TC-B5：历史 56m CEP 误差分解

### 8.1 误差来源汇总

| 编号 | 误差来源 | 量化影响 |
|------|----------|----------|
| E1 | target_n_m 配置不当（目标点在飞机起飞位置附近） | 若 target_n_m=100m，CARP 立即触发，误差 ≈ **50.76m** |
| E2 | pos_n 从 0 积分，CARP 触发时 pos_n ≠ Release_N | 系统偏差 ≈ 0.03m（E3 修正后） |
| E3 | k_drag 符号错误（-0.000637 → +0.000639） | CARP 距离误差 0.03m（修正后影响极小） |
| E4 | CEP 计算口径混淆（around_mean vs to_target） | 系统偏差被 CEP_around_mean 掩盖 |
| E5 | 蒙特卡洛散点使用解析模型（非 JSBSim） | 散布估算不准确 |
| E6 | 投放时刻 pos_n 不等于预设 Release_N | 触发延迟误差 ≈ 0.66m/步 |

### 8.2 根本原因

历史 CEP50_to_target ≈ 56.79m 的**根本原因**是：

当 `target_n_m = 100m` 时，理论释放点 Release_N = 100 - 150.79 = **-50.79m**（负值）。飞机从 pos_n=0 开始飞行，CARP 在 t=0 立即触发（因为 remain_to_target < carp_distance），落点 = 0 + 150.76 = 150.76m，误差 = **50.76m**。

加上蒙特卡洛风场扰动（wind_std=2m/s）产生的散布（CEP50_scatter ≈ 2.72m），总 CEP50 ≈ **50.83m**，与历史实测 56.79m 高度吻合。
