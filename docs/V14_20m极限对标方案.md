# ADS-100 V14 20m 极限对标方案

## 战略决策：为什么继续攻克20m而不是切换到300m？

### 难度对冲
- **20m超低空空投是整个项目的"技术天花板"**
- 如果算法能在20m高度、强地面效应下稳住1.2吨载荷的冲击
- 那么300m任务将只是参数微调的问题

### 物理验证
- 申报书中提到的**"无伞空投精度20米（对地20米）"是项目最硬的指标**
- 300m的有伞空投更多考验的是弹道解算
- **20m考验的是极端的自适应控制律**

### V14策略
将V13的"动压调度"和"积分项"这些高级特性，**反向注入到20m的控制逻辑中**

## V14 核心改进

### 1. 动压与地效耦合增益

**物理背景**：
- 在20m高度，动压q_bar提供基础舵效
- 但地面效应（Ground Effect）会增加升力斜率
- 需要一个复合补偿因子

**公式**：
```
b0_scaled = b0_base * (q_bar / 1200.0) * (1.0 + K_ge)

其中:
K_ge = exp(-h_meas / WingSpan)  # 地效随高度降低而指数级增强
b0_base = 1.6  # 增加基准值以平滑控制
```

**V10的问题**：
- 没有动压补偿
- 没有地效补偿
- b0固定为1.8

**V14的改进**：
- 实时动压补偿：q_factor = q_bar / 1200.0
- 地效补偿：K_ge = exp(-h / WingSpan)
- 复合增益：b0_scaled = b0_base * q_factor * (1.0 + K_ge)

### 2. 带有"积分限幅"的ADRC控制律

**V9/V10的问题**：
- 丢载后由于z3感知不完全
- 往往会停在25m左右（稳态偏高）
- 无法精准回归20.0m

**V14的改进**：
```python
# 积分项（仅在误差较小时开启，防止初始积分饱和）
if abs(e_h) < 3.0:
    sum_e += e_h * dt
    sum_e = clip(sum_e, -10.0, 10.0)
ui = ki * sum_e

# 最终控制律
u0 = kp * e_h + kd * v_up + ui
```

**效果**：
- 即使z3存在残余误差
- 积分项也会缓慢产生一个额外的下压指令
- 强制飞机回归到精准的20.0m

### 3. 自适应动压控制（V13核心特性）

**物理背景**：
- 在丢货瞬间，飞机往往会有俯仰摆动导致空速波动
- 固定增益无法适应动压变化

**V14的改进**：
```python
q_factor = max(q_bar, 100.0) / 1200.0
u_raw = ((u0 - z3) / b0_base) / q_factor + u_ff
```

**效果**：
- 空速快时，减小控制灵敏度，防止振荡
- 空速慢时，增加舵偏角，保证控制力矩

### 4. 温和非对称增益（继承V10）

**V14的设置**：
```python
kp_eff = kp_base * (0.8 if e_h > 0 else 1.2)
```

**优势**：
- 增益比1.5倍（温和）
- 偏高时80%权威
- 偏低时120%权威

### 5. 阻尼墙逻辑（继承V10）

**V14的设置**：
```python
kd_eff = kd_base * (3.5 if (h_meas < 20.0 and v_up < 0) else 1.0)
```

**效果**：
- 防止下冲过猛
- 软刹车

### 6. 防御性指令限幅

**V14的设置**：
```python
u_final = clip(trim_elevator + u_delta, -0.6, 0.25)
```

**逻辑**：
- 在20m任务中，我们可以接受由于控制不足产生的缓慢超调
- 但**绝对不能接受由于控制器过猛导致的触地**
- 这种非对称限幅是20m任务的最后保障

### 7. 提升观测器带宽

**V10的设置**：
```
omega_o = 6.0
beta_03 = 216.0
```

**V14的改进**：
```
omega_o = 7.0
beta_01 = 21.0
beta_02 = 147.0
beta_03 = 343.0
```

### 8. 优化控制参数

**V10的设置**：
```
kp_nominal = 1.69
kd_nominal = 2.60
ki = 0.0  # 无积分项
```

**V14的改进**：
```
kp_base = 1.3
kd_base = 2.5
ki = 0.08  # 积分增益，消除静差
```

### 9. 优化前馈系数

**V10的设置**：
```
k_ff = 0.32
```

**V14的改进**：
```
k_ff = 0.35  # 略微提升
```

## V10 vs V14 对比

### V10平衡混合版（接近触地）

```python
# 观测器
omega_o = 6.0
beta_03 = 216.0

# 控制律
kp_nominal = 1.69
kd_nominal = 2.60
ki = 0.0  # 无积分项

# 无动压补偿
b0 = 1.8  # 固定值

# 前馈
k_ff = 0.32

# 限幅
u_final = clip(u_raw, -0.5, 0.5)
```

**问题**：
- 无动压补偿
- 无积分项，稳态偏高
- 最低高度1.1m，非常危险

### V14 20m极限对标版（工业级）

```python
# 观测器
omega_o = 7.0
beta_03 = 343.0

# 控制律
kp_base = 1.3
kd_base = 2.5
ki = 0.08  # 积分增益

# 动压补偿
q_factor = q_bar / 1200.0
b0_scaled = b0_base * q_factor

# 温和非对称
kp_eff = kp_base * (0.8 if e_h > 0 else 1.2)

# 阻尼墙
kd_eff = kd_base * (3.5 if下冲 else 1.0)

# 前馈
k_ff = 0.35

# 防御性限幅
u_final = clip(u_raw, -0.6, 0.25)
```

**优势**：
- 动压自适应补偿
- 积分项消除静差
- 防御性限幅保命

## V14 如何解决V10的顽疾

### 1. 解决稳态偏高问题

**V10的问题**：
- 丢载后停在25m左右
- z3感知不完全

**V14的解决**：
- 引入ki * sum_e积分项
- 即使z3存在残余误差，积分项也会强制回归20.0m

### 2. 解决动压波动问题

**V10的问题**：
- 固定增益b0=1.8
- 无法适应空速波动

**V14的解决**：
- 实时动压补偿：q_factor
- 空速快时减小灵敏度，空速慢时增大舵偏

### 3. 解决触地风险

**V10的问题**：
- 最低高度1.1m
- 对称限幅[-0.5, 0.5]

**V14的解决**：
- 防御性限幅[-0.6, 0.25]
- 下压舵最大0.25，严格限制

## 实施要点

1. **动压与地效耦合增益**：b0_scaled = b0_base * q_factor * (1.0 + K_ge)
2. **积分限幅逻辑**：仅在abs(e_h)<3.0时开启积分
3. **温和非对称增益**：0.8倍 vs 1.2倍
4. **阻尼墙逻辑**：h<20m且v_up<0时，kd*3.5
5. **防御性限幅**：[-0.6, 0.25]
6. **提升观测器带宽**：omega_o=7.0
7. **优化控制参数**：kp=1.3, kd=2.5, ki=0.08
8. **优化前馈**：k_ff=0.35
