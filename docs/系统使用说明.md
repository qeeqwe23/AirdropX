# ADS-100 无人机空投控制系统 - 使用说明

## 系统概述

ADS-100是一个专业的无人机空投控制系统，用于HY100运输机的精准空投任务。系统采用自抗扰控制(ADRC)算法，能够在高度动态变化的条件下保持稳定的飞行高度。

## 核心功能

### 1. 任务配置
- **载重管理**：支持最大1200kg载荷配置
- **投放模式**：支持单件投放和连续投放（4件连投）
- **投放间隔**：可配置0.50s投放间隔
- **目标高度**：支持20m超低空和300m标准高度任务

### 2. 实时监控
- **高度保持**：实时显示当前高度和目标高度
- **俯仰角监控**：显示飞机俯仰姿态
- **控制指令**：显示升降舵控制指令
- **质量重心**：实时显示全机质量和重心位置

### 3. 环境建模
- **高度设置**：20.0m（可配置）
- **风速模拟**：8.5 m/s
- **风向模拟**：270° (西风)

### 4. 性能分析
- **精度统计**：实时计算高度跟踪误差
- **CEP指标**：圆概率误差分析
- **事件日志**：记录系统运行事件

## 控制器版本

系统当前使用**ADRC-V15极限极效版**控制器：

### V15核心特性

1. **全权控制策略**：
   - kp = 3.2（强比例增益）
   - kd = 5.0（强阻尼增益）
   - ki = 0.30（积分增益）

2. **动压自适应补偿**：
   ```
   b0_tuned = b0_base * (q_bar / 1200.0)
   ```
   - 动压越小，输出u越大，补偿低速舵效损失

3. **强前馈补偿**：
   ```
   k_ff = 0.60
   u_ff = k_ff * (delta_m / m_total)
   ```
   - 投放瞬间注入下压前馈，抵消升力突增

4. **高带宽扩张状态观测器(ESO)**：
   ```
   omega_o = 12.0
   beta_01 = 36.0
   beta_02 = 432.0
   beta_03 = 1728.0
   ```
   - 快速观测和补偿扰动

5. **阻尼墙逻辑**：
   ```
   if h < 20m and v_up < 0:
       kd_eff = kd_base * 4.5
   ```
   - 防止触地的最后防线

6. **抗饱和积分**：
   ```
   if abs(u_prev) < 0.6:
       sum_e += e_h * dt
   sum_e = clip(sum_e, -5.0, 5.0)
   ```
   - 防止积分饱和

### V15性能指标

| 指标 | 数值 | 目标 | 状态 |

| 投放后最大偏差 | 38.51m | <4.0m | ⚠️ 超标 |

| RMS误差 | 23.17m | <2.0m | ⚠️ 超标 |

## 系统架构

### 核心模块

1. **JSBSimWrapper** (`core/jsbsim_wrapper.py`)
   - JSBSim飞行动力学引擎封装
   - 6-DOF真实物理仿真
   - HY100飞机模型集成

2. **MassCGManager** (`core/mass_cg_manager.py`)
   - 质量与重心动态管理
   - 支持4个300kg货物点质量
   - 实时更新飞机惯性特性

3. **ADRCControllerV15** (`core/adrc_controller_v15.py`)
   - 自抗扰控制器
   - 扩张状态观测器(ESO)
   - 前馈补偿与动压自适应

4. **CARPCalculator** (`core/carp_calculator.py`)
   - 计算空投释放点(CARP)
   - 弹道解算
   - 风场修正

### UI模块

1. **MainWindow** (`ui/main_window.py`)
   - 主窗口布局
   - 三栏式设计

2. **ConfigPanel** (`ui/config_panel.py`)
   - 任务配置面板
   - 载重管理
   - 环境建模

3. **MonitorPanel** (`ui/monitor_panel.py`)
   - 实时监控面板
   - 高度/俯仰/控制/质量重心曲线

4. **AnalysisPanel** (`ui/analysis_panel.py`)
   - 精度统计分析
   - 事件日志

5. **SimulationThread** (`ui/simulation_thread.py`)
   - 仿真执行线程
   - 与UI解耦

## 安装与运行

### 环境要求

- Python 3.11+
- Ubuntu 22.04 (推荐)

### 依赖安装

```bash
cd ADS-100-Project   # 或者：cd <你的项目目录>/ADS-100-Project
source ads100_env/bin/activate
pip install -r requirements.txt
```

> 路径提示：系统已实现统一路径管理，默认可自动定位项目根目录；如需从任意工作目录运行（或多项目并行/容器部署），可设置环境变量：
>
> ```bash
> export ADS100_PROJECT_ROOT=/absolute/path/to/ADS-100-Project
> ```

### 运行系统

#### 方式1：使用启动脚本
```bash
./run.sh
```

#### 方式2：直接运行Python
```bash
source ads100_env/bin/activate
python3 run_app.py
```

#### 方式3：运行测试脚本
```bash
source ads100_env/bin/activate
python3 tests/test_full_simulation.py
```

## 使用流程

### 1. 启动系统
- 运行`./run.sh`或`python3 run_app.py`
- 系统自动初始化JSBSim引擎和ADRC控制器

### 2. 配置任务参数
- **左侧面板**：配置载重、模式、间隔
- **环境建模**：设置高度、风速、风向

### 3. 开始仿真
- 点击"开始仿真"按钮
- 系统进入实时监控状态

### 4. 执行投放
- 点击"执行投放"按钮
- 系统自动计算CARP并投放物资
- 控制器自动补偿高度变化

### 5. 查看分析结果
- **右侧面板**：查看精度统计
- **事件日志**：查看系统运行记录
- **性能图表**：`data_logs/performance_analysis.png`

## 性能优化建议

### 1. 提升精度
- 增大前馈系数`k_ff`（当前0.60，可尝试0.70-0.80）
- 优化观测器参数
- 引入自适应增益调度

### 2. 降低振荡
- 适度降低控制增益（但会牺牲响应速度）
- 增大变化率限制（但会牺牲控制权威）

### 3. 提升安全性
- 提高阻尼墙系数（当前4.5倍，可提高到5.0倍）
- 降低最低安全高度阈值

## 故障排除

### 1. 系统无法启动
- 检查虚拟环境是否激活
- 检查依赖是否完整安装
- 查看错误日志

### 2. 仿真性能不佳
- 检查控制器参数配置
- 查看`data_logs/performance_analysis.png`
- 分析事件日志

### 3. 飞机触地
- 降低目标高度（从20m降至15m）
- 提高阻尼墙系数
- 增大控制增益

### 4. 振荡过大
- 降低控制增益
- 增大变化率限制
- 降低观测器带宽
