# ADS-100 振荡问题分析与解决方案

## 问题诊断

### 当前问题表现

从性能分析图表可以看出，系统存在严重的控制振荡：

1. **高度振荡幅度过大**: 高度在 -20m 到 +100m 之间大幅度振荡
2. **俯仰角剧烈变化**: 俯仰角在 -90° 到 +90° 之间摆动
3. **控制饱和**: 升降舵指令长期处于饱和状态（+1.0）
4. **失稳坠毁**: 仿真过程中出现 CRASH DETECTED

### 根本原因分析

根据 Bug2Fix 文档分析，振荡问题的根本原因包括：

1. **观测器增益过高**: ESO 增益参数（beta_01, beta_02, beta_03）设置过大，导致高频振荡
2. **控制增益 b0 不匹配**: 固定的 b0 值无法适应动压变化
3. **参数整定方法不当**: 缺乏系统化的参数整定方法

## 解决方案

### 1. 带宽参数化方法

采用带宽参数化（Bandwidth Parameterization）统一整定参数：

**观测器增益整定**:
```
omega_o = 观测器带宽 (建议初始值: 12.0)
beta_01 = 3 * omega_o
beta_02 = 3 * (omega_o^2)
beta_03 = omega_o^3
```

**控制器增益整定**:
```
omega_c = 控制带宽 (通常设为 omega_o / 4)
kp = omega_c^2
kd = 2 * omega_c
```

### 2. 动态 b0 修正

根据实时动压调整控制增益：

```
b0_current = b0_base * (q_bar / q_nominal)
```

这样可以避免：
- 高速飞行时控制过猛
- 低速飞行时控制无力

### 3. 优化前馈补偿

改进投放前馈补偿逻辑：

```
u_ff = -(delta_m * 9.81) / (max(q_bar, 10.0) * k_ff)
```

其中 k_ff = 0.55 为前馈系数。

## 参数调试标准流程

### 步骤 1: 消除静态振荡

**现象**: 平飞时高度上下快速抖动

**对策**:
- 增大 b0_base（例如从 0.65 调至 0.85），降低控制器灵敏度
- 减小 omega_o，滤除高频噪声

### 步骤 2: 抑制投放跳变

**现象**: 物资离机瞬间飞机上蹿超过 4 米

**对策**:
- 增大 omega_o，使观测器更快感知扰动
- 增大 k_ff，增强预判对冲力度

### 步骤 3: 优化恢复速度

**现象**: 丢货后不振荡，但回到目标高度速度太慢

**对策**:
- 增加 omega_c，提升闭环响应速度

## 关键指标校验

| 指标项 | 判定标准 | 目标值 |
|--------|---------|--------|
| 稳态高度误差 | abs(h_ref - h_meas) | < 0.2 m |
| 丢货瞬时峰值 | max(abs(h_peak - h_ref)) | < 1.8 m |
| 姿态恢复时间 | t_settling (误差 < 0.5m) | < 2.0 s |
| 舵机指令平滑度 | max(abs(u_k - u_k_minus_1)) | < 0.05 |

## 实施计划

1. 创建优化版 ADRC 控制器（ADS100ControllerV2）
2. 实现动态 b0 修正逻辑
3. 调整观测器和控制器带宽参数
4. 运行仿真测试验证效果
5. 迭代优化直到满足性能指标
