# ADS-100 V4 能量优先方案

## 问题诊断

V3版本的核心矛盾：

1. **能量状态不足**：速度低导致动压小（648.9 Pa vs 标称1200 Pa）
2. **前馈增益过载**：单位未对齐，导致前馈量-8.2459远超合理范围
3. **长周期振荡**：10秒周期的Phugoid Mode（长周期运动）
4. **舵效不足**：低动压下气动舵效极低，接近失速边缘

**当前速度分析**：
- 动压648.9 Pa对应速度约32 m/s
- 这对3.5吨+1.2吨的飞机来说几乎无法平飞

## V4 核心改进

### 1. 初始能量状态重设

**提高巡航速度至90 m/s (约175 kts)**

**标称动压计算**：
```
q_ref = 0.5 * rho * V^2 
      = 0.5 * 1.225 * 90^2 
      = 4961 Pa
```

**优势**：
- 较高动压提供线性度更好的舵效
- 使飞机远离失速区
- 增强控制权威性

### 2. 配平逻辑修正

**稳健配平序列**：
1. 设置初始条件（100m, 175kts）
2. 预运行1秒（120帧）让动力系统稳定
3. 执行纵向配平（do_trim）
4. 检查配平状态
5. 如果配平失败，手动给定基准舵位-0.05

### 3. 前馈公式单位对齐

**修正后的无量纲前馈**：
```
u_ff = - K_ff * (delta_m / m_total)

其中：
K_ff    : 前馈灵敏度 (0.3 - 0.5)
delta_m : 投放质量 (300.0 kg)
m_total : 投放前全机总质量
```

**改进效果**：
- V3中的-8.2降至约-0.1（舵效的10%）
- 符合工程常识
- 前馈量不超过舵令全量程的20%

### 4. 强化阻尼项

**带垂直速度反馈的控制律**：
```
u_adrc = ((kp * e1 - kd * v_z) - z3) / b0

其中：
v_z : 垂直速度 (向上为正)
```

**作用**：
- 针对10秒长周期振荡
- 显式引入垂直速度反馈
- 增强系统阻尼

### 5. 参数重整定

**降低控制带宽**：
```
omega_o = 6.0  (保持不变)
omega_c = 1.0  (从1.5降至1.0)

观测器增益：
beta_01 = 18.0
beta_02 = 108.0
beta_03 = 216.0

控制增益：
kp = 1.0
kd = 2.0  (强化微分项)

控制量系数：
b0 = 2.0  (从1.0增至2.0，增加阻尼感)
```

### 6. 输出限幅调整

**放宽限幅范围**：
- 从[-0.4, 0.4]放宽到[-0.6, 0.6]
- 配合高动压，提供足够控制余量

## 改进建议

### 检查do_trim结果

如果`fcs/elevator-pos-norm`返回0.0：
- 查看JSBSim是否报错（Trim Failed）
- 配平失败说明初始速度与高度不匹配
- 务必将初始速度设为175 kts

### 前馈量阈值控制

V3中的-8.2是致命的，会让飞机瞬间"扎头"。V4采用`m_drop / m_total`逻辑确保：
- 投放300kg时，前馈量约-0.025
- 投放1200kg时，前馈量约-0.1
- 符合工程常识

### 消除10s长周期振荡

**原因**：动压不足导致

**解决方案**：
- 提高速度（增大q_bar）
- 降低控制带宽omega_c
- 解耦飞机响应频率与长周期运动频率

## 分步测试策略

### 第一步：平飞测试
- 仅进行100m平飞，不投放货物
- 观察高度波动是否在±0.2m

### 第二步：投放测试
- 平飞稳定后，手动触发投放
- 观察高度曲线

## 关键指标校验

| 检查项 | 理想状态 | 调整动作（若失败） |
|--------|---------|-------------------|
| 初始配平舵位 | abs(Elev) > 0.01 | 检查do_trim是否成功 |
| 实时动压 | q_bar > 4000 Pa | 提高ic/ve-kts |
| 前馈指令 | abs(u_ff) < 0.2 | 减小k_ff |
| 高度响应曲线 | 无正弦波形，快速收敛 | 降低omega_c，增加kd |

## 实施要点

**提高初始速度是解决目前所有问题的先决条件**

1. 修改reset.xml，设置速度为175 kts
2. 实现稳健配平序列
3. 修正前馈公式为无量纲形式
4. 添加垂直速度反馈
5. 调整控制参数
6. 分步测试验证
