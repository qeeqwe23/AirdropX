# ADS-100 V16 黄金平衡方案

## 战略转折：从"暴力解决"到"精准调和"

V15版本通过"全权控制"暴力压制了下冲动能，彻底消除了坠毁风险，并将超调量砍掉了一半。这证明了我们的物理方向（**动压补偿 + 强前馈**）是完全正确的。

目前的**4秒周期振荡**是典型的"过阻尼与高增益耦合"产生的诱发振荡。系统由于太想"锁死"高度，导致控制指令在物理惯性面前产生了**相位提前**。

## V16核心策略

**"权威微降、阻尼精调、能量平滑"**

旨在抹平振荡，最终冲击±4m的任务指标。

## 核心数学模型修正

### 1. 黄金增益调度

**V15的问题**：
- kp=3.2太大→产生过大的纠正力矩→自激振荡
- kd=5.0太大→产生过大的阻尼力矩→相位滞后
- ki=0.30略大→积分超调

**V16的改进**：
```
kp = 2.4  (从3.2降至2.4，降低25%，消除自激振荡)
kd = 3.8  (从5.0降至3.8，降低24%，减小相位滞后)
ki = 0.25 (从0.30略微下调，防止积分超调)
```

**保持高带宽观测器**：
```
omega_o = 12.0
beta_01 = 36.0
beta_02 = 432.0
beta_03 = 1728.0
(保持对1.2吨载荷突变的高敏锐度)
```

**修正控制增益估计**：
```
b0_base = 1.2 (从1.0提高至1.2，增加数值弹性)
```

### 2. 动态指令平滑

**V15的问题**：
- u_rate_limit=0.06太大→舵面变化过快→激振力矩

**V16的改进**：
```
u_rate_limit = 0.04 (从0.06降至0.04，降低33%，强制让机体姿态变化更加柔和)
```

**保持动压补偿逻辑**（V15成功核心）：
```
b0_tuned = b0_base * (q_bar / 1200.0)
```

### 3. 维持V15成功的高前馈量

```
k_ff = 0.60 (保持不变)
```

### 4. 调整阻尼墙系数

```
kd_eff = kd * 4.0 (从4.5降至4.0，配合kd降低)
```

### 5. 调整抗饱和积分阈值

```
if abs(u_prev) < 0.5:  (从0.6降至0.5，更严格)
    sum_e += e_h * dt
sum_e = clip(sum_e, -4.0, 4.0)  (从±5.0降至±4.0)
```

### 6. 调整最终限幅

```
u_final = clip(trim_elevator + u_delta, -0.6, 0.3)  (从[-0.7, 0.3]调整为[-0.6, 0.3])
```

## V16针对性改进逻辑说明

### 1. 解决振荡中心偏离（ki修正）

**V15的问题**：
- 振荡中心在40m
- Kp太大导致初始反向力矩过猛
- 积分项没有足够的权重拉回

**V16的解决**：
- 降低Kp，使得积分项`ki * sum_e`在稳态过程中能够占据主导地位
- 将高度稳死在20m基准线上

### 2. 消除4s周期振荡（Gain Reduction）

**物理分析**：
- 振荡周期长说明系统进入了"大幅度低频晃动"区
- 通过将Kp从3.2降至2.4，降低了控制指令的斜率
- 给飞机的物理转动惯量（Iyy）留出了对齐时间

**配合措施**：
- 将u_rate_limit从0.06压低到0.04
- 从源头上禁止了舵面的"瞬间跳变"
- 防止了激振力矩

### 3. 维持"防触地"底线

**策略**：
- 虽然整体增益下降
- 但V16依然保留了kd_eff 4倍阻尼逻辑
- 这意味着当飞机即便出现振荡下冲时
- 依然会有极强的垂直刹车力
- 确保16m的安全裕度不被侵蚀
