# ADS-100 V9 地面保护与非对称阻尼方案

## 问题诊断

### V8的触地问题

**现象**：
- 9.5秒出现GEAR_CONTACT警告
- 高度降至接近0m
- 之后飞机重新爬升到65m

**触地逻辑分析**：

1. **投放后上跃**：1.2吨物资离机→飞机变轻→升力骤增→飞机上跃
2. **控制器纠偏**：检测到高度偏高→输出强烈的下压指令
3. **下冲过猛**：飞机惯性大+离地仅20m→下压指令过猛→动能过大
4. **触地**：飞机在回到20m的过程中动能过大→直接冲向地面

**根本原因**：
- **对称增益**：上升和下降使用相同的kp
- **下冲能量过大**：从高处回归时获得过大的下坠速度
- **缺乏地面保护**：没有针对低高度的特殊保护逻辑

**后期发散原因**：
- **观测器滞后**：ESO对质量突变后的长时扰动（z3）补偿存在相位滞后
- **beta_03偏小**：无法快速追踪持续性扰动

## V9 核心改进

### 1. 非对称高度增益（Asymmetric Gain）

**核心思想**：**"轻压重拉"策略**

**逻辑**：
```python
e_h = h_meas - h_ref

if e_h > 0:  # 飞机偏高
    kp_eff = kp_nominal * 0.4  # 轻压：防止下冲过猛
else:  # 飞机偏低
    kp_eff = kp_nominal * 2.0  # 重拉：地面保护
```

**V8的问题**（对称增益）：
```
高度 > 20m: kp = 0.81
高度 < 20m: kp = 0.81
```
→ 下冲和拉升力度相同 → 下冲能量过大

**V9的改进**（非对称增益）：
```
高度 > 20m: kp_eff = 0.8 * 0.4 = 0.32  (轻压)
高度 < 20m: kp_eff = 0.8 * 2.0 = 1.60  (重拉)
```
→ 下冲缓慢，拉升强力 → 消除触地风险

**物理意义**：
- **高于目标**：缓慢回归，消除下冲惯性
- **低于目标**：全力拉升，严防触地

### 2. 动态指令限幅（Dynamic Command Shaping）

**V8的限幅**：
```python
u_final = clip(u_raw, -0.4, 0.4)  # 对称限幅
```

**V9的改进**：
```python
u_final = clip(u_raw, -0.5, 0.25)  # 非对称限幅
```

**限幅逻辑**：
- **下压舵（正值）**：最大0.25（防止俯冲触地）
- **上抬舵（负值）**：最大-0.5（增强拉升能力）

**安全保护**：
- 即使算法疯狂想要"扎头"
- 物理指令也会被锁死在轻微下压状态
- 这是20m超低空任务的"生命底线"

### 3. 修正ESO观测器滞后

**V8的设置**：
```
beta_03 = 64.0
```

**V9的改进**：
```
beta_03 = 80.0  (从64.0提升至80.0)
```

**效果**：
- ESO观测器对丢载后的质量变化更敏锐
- 能够更早地抵消长时扰动
- 防止9-15秒的高度漂移

### 4. 收紧变化率限制

**V8的设置**：
```
u_rate_limit = 0.02
```

**V9的调整**：
```
u_rate_limit = 0.015  (进一步收紧)
```

**效果**：
- 进一步平滑控制指令
- 防止急剧变化

### 5. 降低前馈系数

**V8的设置**：
```
k_ff = 0.28
```

**V9的调整**：
```
k_ff = 0.20  (降低前馈，依靠ADRC缓慢回归)
```

**效果**：
- 减小投放瞬间的冲击
- 依靠ADRC自然回归

## V8 vs V9 对比

### V8平滑鲁棒版（触地）

```python
# 观测器
beta_03 = 64.0

# 控制律（对称）
kp = 0.81  # 固定值
kd = 1.80
b0 = 2.0

# 前馈
k_ff = 0.28

# 变化率限制
u_rate_limit = 0.02

# 限幅（对称）
u_final = clip(u_raw, -0.4, 0.4)
```

**问题**：
- 对称增益导致下冲过猛
- 触地风险
- 后期发散

### V9地面保护版（安全）

```python
# 观测器
beta_03 = 80.0  # 提升

# 控制律（非对称）
kp_nominal = 0.8
if e_h > 0:
    kp_eff = 0.32  # 轻压
else:
    kp_eff = 1.60  # 重拉
kd = 1.8
b0 = 2.0

# 前馈
k_ff = 0.20  # 降低

# 变化率限制
u_rate_limit = 0.015  # 收紧

# 限幅（非对称）
u_final = clip(u_raw, -0.5, 0.25)
```

**优势**：
- 非对称增益防止下冲
- 地面保护逻辑
- 消除后期发散

## 设计理念

### 从"完美追踪"到"安全第一"

**V8的设计思路**：
- 追求完美的高度跟踪
- 对称的控制策略
- 快速响应

**V9的设计思路**：
- 安全第一，防止触地
- 非对称的控制策略
- "轻压重拉"

### 非对称控制的工程意义

**物理本质**：
- 飞机离地仅20m
- 下冲风险远大于上跃风险
- 必须采用非对称策略

**工程实践**：
- 低空飞行控制中常用非对称增益
- 地面保护是关键安全特性

## 关键验证指标

| 检查项 | 目标值 (V9) | 验证方法 |
|--------|------------|---------|
| 地面接触警告 | GEAR_CONTACT = FALSE | 必须通过，严禁触地 |
| 回归下冲量 | Min_H > 18.5 m | 确保下冲不低于1.5m |
| 俯仰角摆动 | abs(Pitch) < 4.0 deg | 维持平稳性 |
| 9-15s稳定性 | abs(H - 20.0) < 1.0 m | 消除后期漂移 |
| 投放瞬时超调 | Peak_H < 23.0 m | 满足空投高度包络线 |

## 实施要点

1. **非对称比例增益**：高于目标kp*0.4，低于目标kp*2.0
2. **提升beta_03**：从64.0提升至80.0
3. **降低前馈系数**：k_ff从0.28降至0.20
4. **收紧变化率限制**：u_rate_limit从0.02降至0.015
5. **非对称限幅**：下压舵最大0.25，上抬舵最大-0.5
6. **保持V7/V8的符号对齐和状态归零**

## 预期效果

通过非对称控制策略，V9应能彻底解决20m任务中的触地风险，同时消除后期发散，实现"安全第一"的设计目标。
