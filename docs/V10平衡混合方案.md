# ADS-100 V10 平衡混合方案

## V9失败的深度分析

### 为什么V9会出现102m的巨额超调？

**1. 控制权威"软载减速"失效**

- 1.2吨物资离机后，飞机产生的剩余升力加速度极其巨大
- V9在高度偏高时将增益削减到0.4倍
- **控制律输出的"下压"力矩无法抵消货物离机瞬间产生的"抬头"力矩**
- 飞机不是在受控回归，而是在带弹跳的自由爬升

**2. 稳态平衡点偏移**

- ADRC的观测器z3虽然在工作，但由于kp过小
- 系统在110m高度找到了一个新的力矩平衡点：**升力增量 = 弱化的控制力矩**
- 系统"认为"它在努力压头，但实际上由于指令太弱，根本压不下去

**3. 非对称切换的"泵吸"效应**

- 0.4与2.0之间存在5倍的鸿沟
- 当飞机从122m冲回20m瞬间，控制指令发生了巨大的阶跃
- 飞机像撞在了一块极其坚硬的"弹簧床"上
- 随即被猛烈弹回高空，形成了7秒周期的低频震荡

## V10 核心改进

### 设计理念：平衡混合（Balanced Hybrid）

**核心思想**：
- 不再通过剧烈改变增益来防撞地
- 而是通过**"垂直速度阻尼调节"**来实现平滑的地面规避
- **"高比例增益回归 + 地面近区强阻尼刹车"**

### 1. 提升基础控制带宽

**V9的设置**：
```
omega_c = 0.9
kp_nominal = 0.81
kd_nominal = 1.80
```

**V10的改进**：
```
omega_c = 1.3
kp_nominal = omega_c^2 = 1.69  (提升2倍)
kd_nominal = 2.0 * omega_c = 2.60  (提升1.4倍)
```

**效果**：
- 恢复高度回归权重
- 拥有足够的"肌肉力"来对抗丢载后剩余的升力
- 高度不再稳态漂移在110m

### 2. 温和非对称比例增益（Gentle Asymmetry）

**V9的问题**（剧烈非对称）：
```
高于目标: kp_eff = 0.8 * 0.4 = 0.32  (40%)
低于目标: kp_eff = 0.8 * 2.0 = 1.60  (200%)
增益比: 5.0倍
```
→ 控制不连续，"泵吸"效应

**V10的改进**（温和非对称）：
```
高于目标: kp_eff = 1.69 * 0.85 = 1.44  (85%)
低于目标: kp_eff = 1.69 * 1.25 = 2.11  (125%)
增益比: 1.47倍
```
→ 控制连续，足够权威

**优势**：
- 增益比从5.0降至1.47，保证控制连续性
- 偏高时85%权威，保证足够推力压回
- 偏低时125%权威，适度保护地面

### 3. 地面阻尼墙逻辑（Damping Wall）⭐核心创新

**核心思想**：
- V9的教训：减小kp虽然防触地，但让飞机失去了控制目标的能力
- V10的方案：保持kp稳定，但引入动态阻尼

**逻辑**：
```python
if h_meas < 20.0 and v_up < 0:
    # 飞机在目标高度以下且正在下冲
    kd_eff = kd_nominal * 3.5  # 增加3.5倍微分阻尼进行"软刹车"
else:
    kd_eff = kd_nominal
```

**物理意义**：
- 当飞机在回归20m过程中由于惯性冲过头时
- 高额的阻尼项会像"物理刹车"一样迅速抵消下沉动能
- **既保证了飞机能回到20m，又保证了它不会冲向地面**

**"变阻尼"代替"变比例"防触地**：
- V9：变kp → 失去控制能力
- V10：变kd → 保持控制能力，增加阻尼刹车

### 4. 提升观测器带宽

**V9的设置**：
```
omega_o = 4.0
beta_03 = 80.0
```

**V10的改进**：
```
omega_o = 6.0
beta_01 = 3 * 6.0 = 18.0
beta_02 = 3 * 36.0 = 108.0
beta_03 = 6.0^3 = 216.0
```

**效果**：
- 提升扰动追踪速度
- 更快响应质量变化

### 5. 增强前馈补偿

**V9的设置**：
```
k_ff = 0.20
```

**V10的改进**：
```
k_ff = 0.32  (提升60%)
```

**效果**：
- 增强丢载初期的压头力度
- 配合放宽的变化率限制
- 显著降低丢载瞬间的上跃峰值

### 6. 放宽变化率限制

**V9的设置**：
```
u_rate_limit = 0.015
```

**V10的改进**：
```
u_rate_limit = 0.022  (放宽47%)
```

**效果**：
- 允许升降舵在丢载瞬间更迅速地偏转到位
- 配合提升后的k_ff

### 7. 调整b0参数

**V9的设置**：
```
b0 = 2.0
```

**V10的调整**：
```
b0 = 1.8  (略微降低)
```

**效果**：
- 增大控制输出
- 配合提升的kp

### 8. 调整配平备用值

**V9的设置**：
```
trim_elevator = 0.035
```

**V10的调整**：
```
trim_elevator = 0.038  (修正后的经验配平值)
```

### 9. 非对称限幅调整

**V9的限幅**：
```
u_final = clip(u_raw, -0.5, 0.25)
```

**V10的调整**：
```
u_final = clip(u_raw, -0.6, 0.4)
```

**效果**：
- 允许更大的控制权威
- 上抬舵最大-0.6（增强拉升）
- 下压舵最大0.4（允许更强下压）

## V9 vs V10 对比

### V9地面保护版（失衡）

```python
# 观测器
omega_o = 4.0
beta_03 = 80.0

# 控制律（剧烈非对称）
kp_nominal = 0.8
if e_h > 0:
    kp_eff = 0.32  # 40%
else:
    kp_eff = 1.60  # 200%
kd = 1.8  # 固定

# 前馈
k_ff = 0.20

# 变化率限制
u_rate_limit = 0.015

# 限幅
u_final = clip(u_raw, -0.5, 0.25)
```

**问题**：
- 增益比5.0倍，控制不连续
- 控制权威不足，稳态漂移
- "泵吸"效应，长周期振荡

### V10平衡混合版（平衡）

```python
# 观测器
omega_o = 6.0
beta_03 = 216.0

# 控制律（温和非对称 + 阻尼墙）
kp_nominal = 1.69  # 提升2倍
if e_h > 0:
    kp_eff = 1.44  # 85%
else:
    kp_eff = 2.11  # 125%

# 阻尼墙
kd_nominal = 2.60
if h_meas < 20.0 and v_up < 0:
    kd_eff = 9.1  # 3.5倍
else:
    kd_eff = 2.60

# 前馈
k_ff = 0.32  # 提升60%

# 变化率限制
u_rate_limit = 0.022  # 放宽47%

# 限幅
u_final = clip(u_raw, -0.6, 0.4)
```

**优势**：
- 增益比1.47倍，控制连续
- 控制权威充足，消除漂移
- 阻尼墙防触地，平滑刹车
